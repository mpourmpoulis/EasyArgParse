## Datatypes

The following datatypes are supported, though I should note all of the arguments should be expected to be passed by value

### Primitive data types

- int 

- float

- double 

- std::string 


### Utilities

- std::optional<>


### Containers

- std::vector<>

- std::list<>

- std::pair<>

- std::array<>
